OCR¶

class batukh.tensorflow.ocr.OCR(n_classes)[source]¶

This class is used for octical character recognition.

Example

>>> from batukh.tensorflow.ocr import OCR
>>> m = OCR(177)
>>> m.load_data(train_path="/data/",height=32)
>>> m.train(1)
Parameters

n_classes (int) – number of outputs nodes of ocr model.

decode(inputs, from_pred=True, method='gready', merge_repeated=True, table=None, blank_index=None)[source]¶

Decodes the model logits using ctc decoder.

Example

>>> from batukh.tensorflow.ocr import OCR
>>> import tensorflow as tf
>>> m = OCR(177)
>>> m.load_model("/saved_model/")
>>> x = tf.io.read_file("/image.png")
>>> x = tf.io.decode_png(x,channels=1)
>>> y = m.predict(x)
>>> pred = m.decode(y)
Parameters

  • inputs (tensorflow.Tensor) – Input tensor.

  • from_pred (bol,optional) – True if input is return of ( predict. False if input is a tensorflow.SparseTensor. Default: True

  • method (str,optional) – if 'gready' tensorflow.nn.ctc_greedy_decoder used for decoding.if 'beam_search' tensorflow.nn.ctc_beam_search_decoder used. Default: `` ‘greedy’ ``

  • merge_repeated (bol,optional) – Specifes if similar charsters will be merged. Default: True

  • table (tensorflow.lookup.StaticHashTable,optional) – Table according to which maping is done. Default: self.train_dl.inv_table

  • blank_index (int,optional) – Blank index. Default : self.train_dl.blank_index

Returns

decoded list of strings

Return type

list
