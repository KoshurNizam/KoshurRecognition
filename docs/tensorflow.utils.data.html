tensorflow.utils.data¶

class batukh.tensorflow.utils.data.dataloader.SegmentationDataLoader(path, n_classes)[source]¶

Loads the tensorflow.data.Dataset for PageExtracter, ImageExtracter, LayoutExtracter and BaselineDetecter classes.

Example

>>> from batukh.tensorflow.utils.data.dataloader import SegmentationDataLoader
>>> dl=SegmentationDataLoader("/data/",2)
>>> for i,j in dl(batch_size=1):
...     print(i.shape,j.shape)
...     break
Parameters

  • path (str) – Path of the folder containing originals folder and labels folder to be loaded in dataset. Folder names must be as mentioned.

  • n_classes (int) – number of classes in label images.

class batukh.tensorflow.utils.data.dataloader.OCRDataLoader(path, height=64)[source]¶

Loads the Dataset for OCR class.

Example .. code:: python

>>> from batukh.tensorflow.utils.data.dataloader import OCRDataLoader
>>> dl=OCRDataLoader("/data/",64)
>>> for i,j in dl(1,1):
...     print(i.shape,j.values)
...     break
Parameters

  • path (str) – Path of folder containing images folder,labels.txt and table.txt to be loaded in dataset.Name of folders and files should be same as mentioned.

  • height (int,optional) – Specifies the height to which all images will be resized keeping same aspect ratio. Default: \(64\)

decode(inputs, from_pred=True, method='gready', merge_repeated=True)[source]¶

Decodes the model logits using ctc decoder.

Example

>>> from batukh.tensorflow.ocr import OCR
>>> import tensorflow as tf
>>> m = OCR(177)
>>> m.load_model("/saved_model/")
>>> x = tf.io.read_file("/image.png")
>>> x = tf.io.decode_png(x,channels=1)
>>> y = m.predict(x)
>>> pred = m.train_dl.decode(y)
Parameters

  • inputs (tensorflow.Tensor) – Input tensor.

  • from_pred (bol,optional) – True if input is return of ( predict. False if input is a tensorflow.SparseTensor. Default: True

  • method (str,optional) – if 'gready' tensorflow.nn.ctc_greedy_decoder used for decoding.if 'beam_search' tensorflow.nn.ctc_beam_search_decoder used. Default: `` ‘greedy’ ``

  • merge_repeated (bol,optional) – Specifes if similar charsters will be merged. Default: True

Returns

decoded list of strings

Return type

list
